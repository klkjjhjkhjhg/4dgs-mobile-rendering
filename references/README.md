# References

> 参考文献库。每条文献一个 markdown 文件,记录完整信息 + 关键 takeaway。

## 命名约定

`<年份>-<第一作者>-<方法名>.md`,例:
- `2024-wu-4dgs.md`
- `2025-4dgs-1k-lite.md`
- `2023-yang-dynamic-3dgs.md`

## 每条文献必含

- **标题 / 作者 / 机构 / 会议或期刊 / 年份 / arxiv 链接 / 项目页 / 代码仓库**
- **核心方法 3~5 条**
- **关键数字**(精度 / FPS / 资源体积 / 训练时长)
- **与本调研的关系**(对标 / 借鉴 / 反例)
- **代码仓库中的关键文件引用**(用于后续 prototype 复用)

## 类别(建议标签)

- `[high-precision]` 高精度 4DGS 表示相关
- `[acceleration]` 渲染加速相关
- `[mobile]` Android / 移动端 GPU 实现相关
- `[vulkan]` Vulkan / GPU 加速范式相关
