\section{Datasets \& Metrics}

\subsection{常用数据集}
4DGS 训练 / 评估常用的动态数据集包括:
\begin{itemize}
  \item \textbf{D-NeRF} (Synthetic): 8 个合成场景, 简单形变
  \item \textbf{DyCheck} (iPhone): 真实动态场景, 移动端采集
  \item \textbf{Nerfies / HyperNeRF}: 单目动态人体 / 物体
  \item \textbf{Plenoptic Video} (Google): 12 相机阵列, 高质量
  \item \textbf{Meet Room / CMU Panoptic}: 多视角多人交互
\end{itemize}

\subsection{评估指标}
\begin{itemize}
  \item \textbf{质量}: PSNR / SSIM / LPIPS (per-frame)
  \item \textbf{时序一致性}: Temporal LPIPS / Flow warping error
  \item \textbf{速度}: FPS (decode + render) / 训练时间 (min)
  \item \textbf{存储}: 模型大小 (MB) / 显存峰值 (MB)
  \item \textbf{移动端特有}: 功耗 (mW) / 热稳定性 (sustained FPS)
\end{itemize}
