\section{Conclusion}

本综述系统性整理了 2023--2026 H1 移动端实时 4DGS 方向的 50+ 篇代表性工作,
覆盖表示 / 训练加速 / 渲染加速 / 移动端部署四大主题。当前 Snapdragon 8 Gen 4
平台已能实现桌面级 3DGS 实时渲染, 但 4DGS 仍受限于显存带宽 / Vulkan 驱动成熟度
/ 功耗墙三重约束。

后续研究方向包括 feed-forward 4DGS、硬件-软件协同设计、4DGS ray tracing、
时序压缩标准、跨模态融合等。本综述与项目 README 互补, 前者侧重学术调研纵深,
后者侧重工程落地路径。
