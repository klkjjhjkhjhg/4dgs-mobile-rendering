\section{Mobile Deployment}

\subsection{当前现状}
Snapdragon 8 Gen 4 + Adreno 830 是 2026 H1 移动端最强 SoC 之一,
FP32 算力 ~2 TFLOPs, Vulkan 1.3 驱动成熟。已部署的 4DGS 工作:

\begin{itemize}
  \item Mobile-GS / Flux-GS: 桌面 3DGS 实时,4D 仍在探索
  \item Lumina: 移动端 neural rendering 标杆
  \item GS-NFS: Jetson Orin (边缘设备) 4DGS decode
\end{itemize}

\subsection{核心瓶颈}
\begin{enumerate}
  \item \textbf{显存带宽}: 4DGS 的时空联合高斯数远大于静态 3DGS,
        bandwidth-limited 严重
  \item \textbf{Vulkan 1.3 驱动成熟度}: compute shader 优化、
        subgroup operations、tile-based rendering 兼容
  \item \textbf{功耗墙}: 移动端 thermal throttling 导致持续渲染掉帧
  \item \textbf{动静态分离的存储开销}: 即使分离后,动态部分仍占大头
\end{enumerate}
