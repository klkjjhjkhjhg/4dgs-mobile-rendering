# docs/miscs/

> **目录语义**: 外部站点 / 单点调研 / 项目外的零散归档. 跟 `discovery/` (项目内横向调研), `china-landscape/` (机构调研), `survey/` (出版综述) 是平行的归档位.

## 命名规范

`{topic}-{subtopic-slug}-{YYYY-MM-DD}.md`, 例如 `4dv-ysjf-render-arch-2026-07-22.md`.

## 现有归档

## 与其它目录的边界

- 不是项目内调研 → 用 `docs/discovery/`
- 不是机构 / 单位 landscape → 用 `docs/china-landscape/`
- 不是综述 / 论文 note → 用 `docs/survey/` 或 `docs/appendix/paper-notes/`
- 演进图 / 可视化 → 用 `docs/evolution/`
- 项目任务规划 / 进度 → 用 `docs/mission/`
- 规范 / 路线图 → 用 `docs/01-` 风格编号文档
